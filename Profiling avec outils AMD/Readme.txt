3 outils de profiling AMD

- rocprofv3 pour les activités GPU uniquement (equivalent Nvidia nvprof)

- rocprof-sys pour les activités CPU ET GPU (equivalent Nvidia nsight system)

- rocprof-compute pour les données hardware/counters d'un ou des 10 premiers kernels (equivalent Nvidia nsight compute)

voir details dans le fichier fourni pour chaque outil.
